# React Three Fiber Game Dev Recipes

A collection of minimal patterns and techniques for 3D game development with React Three Fiber.

## Skills

- [Bone Attachment](./bone-attachment/SKILL.md) - Attach meshes to character bones
- [Camera Shake](./camera-shake/SKILL.md) - Screen shake effect with exponential decay
- [Damage Numbers](./damage-numbers/SKILL.md) - Floating damage numbers with CSS animations
- [Health Bars](./health-bars/SKILL.md) - CSS-styled health bars in 3D space
- [Koota ECS](./koota/SKILL.md) - Data-oriented ECS with React bindings
- [Low Health](./low-health/SKILL.md) - Pulsing red vignette overlay
- [Miniplex ECS](./miniplex/SKILL.md) - Minimalistic ECS with TypeScript support
- [Pushback](./pushback/SKILL.md) - Enemy knockback with flash effect
- [Reactive Polling](./reactive-polling/SKILL.md) - Poll for state changes with useReactive
- [Smooth Interpolation](./smooth-interpolation/SKILL.md) - Exponential smoothing for animations
- [UI useFrame](./ui-useframe/SKILL.md) - Sync UI outside Canvas with render loop
- [Weapon Trail](./weapon-trail/SKILL.md) - Ribbon trail effect with BufferGeometry
- [Zustand](./zustand/SKILL.md) - Simple state store for entity management

## Stack

- React Three Fiber v10 (WebGPU)
- Three.js TSL for shaders
- React 19 with React Compiler
- Next.js Pages Router
